# Olist E-commerce SQL Analysis

## Project Overview

This project analyzes the Olist e-commerce dataset using PostgreSQL to answer key business questions related to sales, products, customers, and sellers.

## Tools

- PostgreSQL
- SQL

## Business Questions

- What is the total revenue?
- Which states generate the highest revenue?
- Which product categories perform best?
- Which payment methods most frequently used?
- Who are the top sellers?
- How do reviews relate to delivery performance?

## SQL Concepts Used

- INNER JOIN
- GROUP BY
- CTE: Common Table Expressions (CTEs)
- CASE WHEN
- Window Functions (RANK)
- Aggregate Functions
- SUM, COUNT, AVG

## Key Insights

This analysis identifies top-performing product categories, customer purchasing behavior, seller performance, payment preferences, and regional sales trends.
